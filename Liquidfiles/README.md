This script will automatically send liquidfiles to the recipients with the correct details

How to use:
1. open the .ps1 file, in the apikey variable place the your liquidfiles api token from settings > account settings > api inthere
2. open the xlsx file and enter the fields as labelled, the only required field is "requester email"
3. export as files names users.csv
4. either download vscode and run the script in vscode or copy the contents of the .ps1, right click the folder that it is placed in and click, "Open in Terminal", then paste the contents of the file in the terminal.

(DO NOT CHANGE THE NAMES OF THE HEADINGS)
